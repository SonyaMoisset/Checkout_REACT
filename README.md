# A React web application for a Checkout Form

To see the project:
- Clone/download it
- Use ```npm install```
- Use ```npm start```

## This application includes examples of major features of React including:
- ES6 syntax
- Templating string
- Destructuring assignment
- Component-based architecture
- CSS Partials/Imports
